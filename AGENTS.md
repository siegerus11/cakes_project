# Cakes Project - Инструкции для агента

## Контекст проекта

**Cakes** — это веб-приложение для кондитерской, написанное на React с использованием Vite.
Проект представляет собой каталог тортов и бенто-тортов с возможностью оформления заказа.

### Основной функционал:

-   Каталог тортов и бенто-тортов с фильтрацией и сортировкой
-   Карточка товара с выбором начинки, веса и дополнений
-   Корзина заказов
-   Оформление заказа
-   Страницы: главная, о нас, каталог, доставка, контакты, FAQ

## Стек технологий

### Основной фреймворк:

-   **React 18** — UI библиотека
-   **TypeScript 4.8** — типизация
-   **Vite 3** — сборщик и dev-сервер

### State Management:

-   **Redux Toolkit** — глобальное состояние
-   **React Redux** — интеграция с React

### Роутинг:

-   **React Router DOM 6** — навигация

### Стилизация:

-   **SASS (SCSS)** — основной препроцессор стилей
-   **CSS Modules** — изоляция стилей компонентов
-   **Tailwind CSS 3** — утилитарные CSS-классы (настроен, но используется ограниченно)
-   **@emotion/react, @emotion/styled** — CSS-in-JS (настроен, но основное использование — SCSS)

### Карты:

-   **Leaflet** — интерактивные карты
-   **React Leaflet** — React-обертка для Leaflet

### Линтинг и форматирование:

-   **ESLint** (конфиг Airbnb) — линтинг JS/TS
-   **Prettier** — форматирование кода
-   **Stylelint** — линтинг SCSS

### Тестирование:

-   **Jest 29** — тестовый фреймворк
-   **Testing Library** — тестирование React-компонентов

### Другие зависимости:

-   **prop-types** — runtime проверка типов
-   **react-error-boundary** — обработка ошибок
-   **react-draggable** — перетаскивание элементов
-   **uuid** — генерация уникальных ID

│   ├── cake-offers-data/  # Данные о тортах

## Архитектура

### Redux Store:

-   `NameSpace.Data` — данные о тортах (cake-offers-data)
-   `NameSpace.Main` — основной процесс (сортировка)
-   `NameSpace.Cart` — процесс корзины (cart-process)

### Маршруты (AppRoute):

-   `/` — Главная
-   `/about` — О нас
-   `/catalog` — Каталог
-   `/catalog/cakes` — Каталог тортов
-   `/catalog/bento-cakes` — Каталог бенто-тортов
-   `/cake-offer/:id` — Карточка торта
-   `/contacts` — Контакты
-   `/delivery` — Доставка и оплата
-   `/shopping-cart` — Корзина
-   `/order-registration` — Оформление заказа
-   `/thanks-page` — Страница благодарности

### Ключевые типы:

-   `CakeOffer` — объект торта с начинками, весом, дополнениями
-   `CakeOrder` — объект заказа
-   `Filling` — начинка
-   `Optional` — дополнительные опции

## Соглашения по коду

### Стиль кода:

-   Используй функциональные компоненты с хуками
-   Типизация через TypeScript (не prop-types, если не требуется)
-   Именование компонентов в PascalCase
-   Именование файлов компонентов в kebab-case (например, `main-page.tsx`)
-   Стили компонентов в отдельном `.module.scss` файле

### Импорты:

-   Абсолютные импорты от `src/` (через `../../`)
-   Порядок: React → внешние библиотеки → внутренние модули → стили

### Стилизация:

-   Основной подход: CSS Modules + SCSS
-   Глобальные стили в `global.module.scss`
-   Переменные в `_variables.scss`
-   Миксины в `_mixins.scss`
-   Брейкпоинты: 1200px, 992px, 767px, 576px

## Важные замечания

1. **Язык контента**: Русский (ru-RU)
2. **Валюта**: Рубли (₽)
3. **Формат цен**: Используй утилиту `getFormattedPrice()` для форматирования
4. **Изображения**: Хранятся в `public/images/`
5. **Шрифты**: Хранятся в `public/fonts/`, подключение через `_fonts.scss`
6. **Иконки**: SVG-спрайт, использование через `<use xlinkHref="#icon-name">`
